# Average Revenue Per User (ARPU)

Average revenue per user measures how much revenue you’re generating for each active customer. [^1] [^4] [^5] [^6] [^2] [^3]

## Calculation

$$
\textrm{ARPU} = \frac{\textrm{MRR}}{\textrm{total number of active users in a month}} [^1]
$$
